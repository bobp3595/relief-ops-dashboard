Connects this dashboard to a Google Sheet as a live database. Every intake, status update, and metric writes instantly to your sheet. Your whole team can view it in real time.

Setup guide — 5 minutes
  1. Go to console.cloud.google.com → create a project named Relief Ops
  2. Search Google Sheets API → Enable it
  3. APIs & Services → Credentials → Create Credentials → OAuth client ID
  4. Configure consent screen if prompted: External, add your email, save through all steps
  5. Application type: Web application
  6. Authorized JavaScript origins: add your Netlify URL (e.g. https://yourapp.netlify.app)
  7. Copy the Client ID, paste above, click Save & Connect
  8. Leave Sheet ID blank — a new sheet is created automatically in your Google Drive
  9. Share that sheet with your team so they can view live data
